使用自定义字体

vue 的 scoped 原理

Scoped的原理是通过在组件的根元素上添加一个唯一的属性来实现的。这个属性的值是一个随机的字符串，它会被添加到组件中的每个元素上。这样，组件中的所有样式都会被限制在这个属性的值所代表的作用域内

类数组概念以及如何转换为数组

类数组概念

  • 类数组是一个对象
  • 属性名称即键值使用数字
  • 带有 length 属性

类数组特点

  • 可以转化为真正的数组
  • 没有数组方法，但不可以调用数组的方法，如 push、concat 等
  • 可以像数组那样循环

常见的类数组

  • arguments
  • HTMLCollection（元素集合）：比如 ele.children、比如 document.getElementByTagName、document.getElementByClassName 等等
  • NodeList：ele.childNodes，还有类似于 document.querySelectorAll 等方法也可以返回类数组对象
  • string、LocaStorage 和 sessionSotrage 也是

类数组转化为数组的方法

  • Array.from()
  • 扩展运算符：[…str]、[…ele.childNodes]
  • slice 方法以及 Array.prototype.push.call()、Array.prototype.splice.call()、Array.prototype.apply.call()

行内元素设置为浮动元素后可以设置宽高原因

因为浮动后会给元素生成块级框（即实现模式转换）

JS 获取和设置盒模型对应的宽和高的四种方法

  • dom.style.width/height：这种方法只能获取到内联样式的宽和高，而样式一般不回以内联方式写，所以这种方式不适用现在大多数的开发
  • dom.currentStyle.width/height：这种方式只适用于 IE，直接 PASS
  • window.getComputedStyle(dom).width/height：getComputedStyle()是一个可以获取当前元素所有最终（划重点）使用的 CSS 属性值的方法；兼容性很好
  • dom.getBoundingClientRect().width/height：getBoundingClientRect 用于获取某个元素相对于视窗的位置集合。集合中有 top, right, bottom, left,height,width,x,y 共 8 个属性；兼容性很好
  • dom.offsetWidth/offsetHeight​​  包括高度（宽度）、内边距和边框，不包括外边距。最常用，兼容性最好

js 中转成数字类型的方法

  • parseInt
  • parseFloat()
  • Number()
  • 使用一元运算符(+)
  • Math.floor()
  • 乘以数字
  • 双波浪号（~~）运算符

无法转换的会输出 NaN

Number

  • 1.如果是纯数字的字符串，则直接将其转换为数字
  • 2.如果字符串中有非数字的内容，则转换为 NaN
  • 3.如果字符串是一个空串或者是一个全是空格的字符串，则转换为 0

setTimeout 实现 SetInterval

  • setInterval 存在的问题：执行的时间稍有延迟，原因：是在 EventLoop 中，setInterval 的执行机制造成的。每个 setTimeout 产生的任务会直接 push 到任务队列中；而 setInterval 在每次把任务 push 到任务队列前，都要进行一下判断(看上次的任务是否仍在队列中，如果有则不添加，没有则添加)
  • setInterval 的缺点：1、使用 setInterval 时，某些间隔会被跳过（如果上一次执行代码没有执行，那么这次的执行代码将不会被放入队列，会被跳过）2、可能多个定时器会连续执行（上一次代码在队列中等待还没有开始执行，然后定时器又添加第二次代码，第一次代码等待时间和执行时间刚好等于第二次代码执行）
